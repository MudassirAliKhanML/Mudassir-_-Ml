# Heart Disease Prediction using Machine Learning ❤️

### Project Description
یہ ایک Machine Learning Project ہے جو Cleveland Heart Disease ڈیٹا سیٹ پر بنایا گیا ہے۔
Logistic Regression Model استعمال کر کے یہ پیش گوئی کرتا ہے کہ مریض کو Heart Disease کا رسک ہے یا نہیں۔

### Model Performance
- *Algorithm*: Logistic Regression
- *Accuracy*: 88%
- *Dataset*: heart_disease_cleveland.csv - 303 patients, 13 features

### Features Used
Model یہ 13 چیزیں چیک کرتا ہے:
1. age - عمر
2. sex - جنس (0 = عورت, 1 = مرد)
3. cp - سینے کا درد کی قسم (0-3)
4. trestbps - آرام کا بلڈ پریشر
5. chol - کولیسٹرول
6. fbs - فاسٹنگ بلڈ شوگر > 120؟
7. restecg - ECG رزلٹ (0-2)
8. thalach - زیادہ سے زیادہ دل کی دھڑکن
9. exang - ورزش سے درد؟
10. oldpeak - ST depression
11. slope - ST slope (0-2)
12. ca - بڑی شریانیں 0-3
13. thal - تھلیسیمیا (0-3)

### How to Use
1. Heart_Disease_Prediction.ipynb کھولو
2. Run All Cells دباؤ
3. آخر میں نئے مریض کا ڈیٹا ڈال کر Prediction چیک کرو

### Saved Model
heart_model.pkl فائل میں تیار Model سیو ہے۔ اسے لوڈ کر کے بغیر ٹریننگ کے Prediction لے سکتے ہو۔
